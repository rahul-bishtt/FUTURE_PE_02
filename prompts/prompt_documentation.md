## Prompt 1 — Brand & Audience Definition

**Goal:** Establish the brand identity and target persona before generating content

```
You are an expert D2C brand strategist specializing in Indian skincare.

Create a complete brand profile for a fictional Indian D2C skincare brand called "GlowLab" 
with the following details:
- Hero product: Vitamin C Face Serum (15% Vitamin C + Hyaluronic Acid)
- Price: ₹699
- Target market: Indian consumers aged 18–35

Include:
1. Brand positioning statement (1–2 sentences)
2. Primary target audience persona (name, age, city, skin concern, buying behavior)
3. Secondary target audience persona
4. 5 brand voice pillars (honest, conversational, etc.)
5. List of words/phrases to use vs. avoid in all content

Format with clear headers and bullet points.
```

**Prompt Engineering Technique:** Role prompting + structured output request + constraint setting
**Output Generated:** Brand overview + 2 audience personas + voice pillars + vocabulary guide

## Prompt 2 — Hook Generation

**Goal:** Generate 12 diverse, high-performing hooks for Instagram Reels

```
You are a performance marketing copywriter specializing in UGC (User Generated Content) ads 
for Instagram Reels. 

Write 12 scroll-stopping hooks for a UGC ad promoting GlowLab Vitamin C Face Serum.

Product details:
- Name: GlowLab Vitamin C Face Serum
- Price: ₹699
- Key benefit: Reduces pigmentation, dark spots, and dullness
- Target audience: Indian women and men aged 18–35

Rules for each hook:
- Must be under 20 words
- Must sound like a real person, not an ad
- Must create curiosity or emotional resonance
- Must be written in first person ("I") where applicable

Organize hooks into 4 types:
1. Pain Point Hooks (3 hooks)
2. Question Hooks (3 hooks)
3. Shock / Stat Hooks (3 hooks)
4. Relatable / POV Hooks (3 hooks)

Include Indian cultural references (chai, Swiggy, college life, Indian sun) where natural.
```

**Prompt Engineering Technique:** Role prompting + constraint setting + categorization request + cultural context injection
**Output Generated:** 12 hooks across 4 types with performance guide

## Prompt 3 — UGC Ad Scripts

**Goal:** Generate 5 full Instagram Reels scripts in authentic UGC tone

```
You are an expert UGC content creator and ad copywriter for Indian D2C brands.

Write 5 UGC-style ad scripts for Instagram Reels promoting GlowLab Vitamin C Face Serum.

Brand context:
- Product: Vitamin C Face Serum (15% Vit C + Hyaluronic Acid)
- Price: ₹699
- Brand voice: Honest, conversational, Indian and relatable, credible without being clinical
- Target: Indian women/men 18–35 with pigmentation or dull skin concerns

Script requirements:
- Each script: 15–30 seconds when spoken at natural pace (approx 60–90 words)
- Structure each script: HOOK → PROBLEM → SOLUTION → PROOF → CTA
- Sound like a real user giving an honest review, NOT a brand advertisement
- Include visual direction in [VISUAL] tags
- Include spoken words in [VO] tags
- Each script must use a different angle:
  1. Script 1: Skeptic who spent too much on other serums
  2. Script 2: Educational / problem-solution angle
  3. Script 3: Emotional confidence transformation story
  4. Script 4: Ingredient education (science made simple)
  5. Script 5: POV / casual Gen Z style

Include a summary table at the end comparing all 5 scripts by angle, target viewer, and emotional trigger.
```

**Prompt Engineering Technique:** Role prompting + chain structure (HOOK→PROBLEM→SOLUTION→PROOF→CTA) + angle specification + persona-based content + output formatting
**Output Generated:** 5 complete UGC scripts with visual directions and comparison table

## Prompt 4 — Captions & CTAs

**Goal:** Generate platform-specific captions, CTA variations, and hashtag strategy

```
You are a social media copywriter specializing in Instagram content for Indian D2C brands.

Create a complete caption and CTA pack for GlowLab Vitamin C Face Serum Instagram Reels.

Requirements:
1. Write 5 caption sets — one for each of these scripts:
   - Skeptic converted
   - Problem-solution  
   - Confidence story
   - Ingredient explainer
   - POV/Casual

   For each caption set, provide:
   - A short caption (2–3 lines + hashtags)
   - A long caption (5–8 lines + hashtags)

2. Write 4 categories of CTA variations:
   - Soft CTAs (4 options)
   - Direct CTAs (4 options)
   - Engagement CTAs (4 options)
   - Urgency CTAs (4 options)

3. Create a hashtag strategy:
   - Primary hashtags (always use)
   - Secondary hashtags (mix and match)
   - Trending/engagement hashtags

Format everything with clear headers. Include a best-practice note on hashtag usage.
Maintain GlowLab's brand voice: honest, conversational, Indian and relatable.
```

**Prompt Engineering Technique:** Structured multi-part output request + brand voice constraint + categorization + platform specificity
**Output Generated:** 5 caption sets (short + long) + 16 CTA variations + hashtag strategy

## Prompt 5 — Brand Voice Guide

**Goal:** Create a comprehensive brand voice document for content consistency

```
You are a brand strategist creating a brand voice guide for GlowLab — an Indian D2C skincare brand.

Create a detailed brand voice guide for their Instagram UGC ad content. Include:

1. Brand overview (2–3 sentences)
2. Two detailed audience personas with: name, age, location, skin concern, buying behavior, and emotional trigger
3. Five brand voice pillars — each with:
   - Pillar name and emoji
   - 2–3 sentence description
   - One "DO" example and one "DON'T" example
4. Tone guide across content types (hooks, scripts, captions, ingredient posts)
5. Words & phrases TO USE (10+ examples)
6. Words & phrases to AVOID (10+ examples with reason)
7. Instagram Reels posting best practices (format as a table)

Make the guide practical — something a content creator could pick up and immediately understand 
how to write for GlowLab without ever having spoken to the founders.
```

**Prompt Engineering Technique:** Role prompting + structural scaffolding + practical use-case framing + table formatting request
**Output Generated:** Full brand voice guide with personas, pillars, vocabulary guide, and posting tips
