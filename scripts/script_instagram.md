# 📜 Instagram Reels UGC Ad Scripts — GlowLab Vitamin C Face Serum
**Platform:** Instagram Reels
**Format:** 15–30 seconds | Conversational | Authentic UGC tone
**Structure:** HOOK → PROBLEM → SOLUTION → PROOF → CTA

---

## How to Read These Scripts
- **[VISUAL]** = What is shown on screen
- **[VO]** = Voiceover / what the person says
- Each script is 15–30 seconds when spoken at natural pace
- Written to sound like a real user, not a paid actor

---
