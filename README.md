# 📱 Task 2 — AI Content Marketing using UGC Ads
**Future Interns | Prompt Engineering Track | Task 02**

## 📋 Overview
This project is a complete AI-generated **UGC Ad Content Pack** for **GlowLab** — a fictional Indian D2C skincare brand. The pack was created using structured prompt engineering with Claude AI.

## 🎯 Product Details
- **Brand:** GlowLab
- **Product:** Vitamin C Face Serum (15% Vitamin C + Hyaluronic Acid)
- **Price:** ₹699
- **Target Audience:** Indian women & men aged 18–35 with dull skin, pigmentation, or dark spots
- **Platform:** Instagram Reels
- **Brand Type:** D2C (Direct to Consumer)
